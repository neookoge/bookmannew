初めてのHTML & CSS

投稿日: 2025年6月22日 | カテゴリ: Web開発

今日はHTMLとCSSの基本について学びました。HTMLはウェブページの構造を作り、CSSは見た目を美しくするために使われます。このページも、まさにその成果です！

HTMLとは？

HTML (HyperText Markup Language) は、ウェブコンテンツの構造を定義するためのマークアップ言語です。見出し、段落、画像、リンクなどを要素として記述します。

CSSとは？

CSS (Cascading Style Sheets) は、HTMLで作成されたウェブページの見た目（色、フォント、レイアウトなど）を装飾するためのスタイルシート言語です。CSSを使うことで、ウェブページをより魅力的で使いやすいものにできます。
