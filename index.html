プライバシーポリシー

当アプリは、ユーザーのプライバシーを最優先に考えます。本プライバシーポリシーは、当アプリが情報の収集を行わないこと、および使用する外部サービス（Google AdMob、Google Books API）が情報をどのように利用するかについて説明します。

1. 当アプリによる個人情報の非収集

当アプリは、氏名、メールアドレス、電話番号、位置情報など、ユーザー個人を特定できる情報を収集しません。また、アプリの利用状況に関するデータ（例：アプリの起動回数、特定の機能の利用頻度など）についても、個人を特定できない範囲で、かつ匿名化された形式でのみ内部的に利用し、外部に送信することはありません。

2. 第三者サービスによる情報の利用

2.1. Google AdMob（広告配信）

本アプリは、広告表示のためにGoogleが提供する広告配信サービス AdMob を利用しています。AdMobは、広告のパーソナライズ、広告の表示回数制限、広告のパフォーマンス測定などを目的として、以下の情報を自動的に収集する場合があります。これらの情報は、ユーザー個人を直接特定できるものではありません。

  • デバイス情報：デバイスモデル、オペレーティングシステム、IPアドレス、ブラウザの種類、言語設定。
  • 広告識別子：デバイスの広告ID（Android Advertising ID）。これは、ユーザーの興味関心に基づいた広告をパーソナライズするために使用されます。
  • 利用状況データ：アプリの利用状況、広告の表示回数、クリック数、アプリ内での広告関連の操作。
  • おおよその位置情報：ユーザーがデバイスの位置情報サービスの利用を許可している場合、おおよその位置情報が広告配信の目的に利用されることがあります。
これらの情報は、広告のパーソナライズ、広告の最適化、広告詐欺の検出および防止のために利用されます。

2.1. Google AdMob（広告配信）

本アプリは、書籍情報の検索および表示のために Google Books API を利用しています。Google Books APIは、書籍のタイトル、著者、概要、カバー画像、プレビューなどの公開されている情報を取得するために使用されます。 このAPIを利用する過程で、GoogleはAPIリクエスト元に関する基本的な情報（例：匿名のIPアドレス、リクエストの種類）をログとして記録する場合があります。これらの情報は、APIサービスの提供、維持、改善、および不正利用の防止のために利用されます。

3. データ共有について

上記2.1および2.2に記載のサービスによって収集される情報は、Google AdMob および Google Books API を含む Google のサービスと共有されます。Google が収集したデータの利用については、Google のプライバシーポリシー をご参照ください。 [Google プライバシーポリシー]
